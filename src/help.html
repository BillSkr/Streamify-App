Βοήθεια & Οδηγίες Χρήσης

Πώς δημιουργώ έναν λογαριασμό;

Βήμα 1: Κάντε κλικ στο κουμπί "Εγγραφή" στην αρχική σελίδα ή στο μενού πλοήγησης.

Βήμα 2: Συμπληρώστε όλα τα απαιτούμενα πεδία:

  • Όνομα και Επώνυμο
  • Username (μοναδικό)
  • Email (μοναδικό)
  • Κωδικός πρόσβασης

Βήμα 3: Κάντε κλικ στο κουμπί "Εγγραφή" για να ολοκληρώσετε τη διαδικασία.

Σημείωση: Βεβαιωθείτε ότι το username και το email που επιλέγετε δεν χρησιμοποιούνται ήδη από άλλον χρήστη.

Πώς συνδέομαι στον λογαριασμό μου;

Βήμα 1: Κάντε κλικ στο κουμπί "Σύνδεση" στην αρχική σελίδα.

Βήμα 2: Εισάγετε το username ή email και τον κωδικό πρόσβασής σας.

Βήμα 3: Κάντε κλικ στο κουμπί "Σύνδεση".

Προβλήματα σύνδεσης: Αν αντιμετωπίζετε προβλήματα, ελέγξτε ότι έχετε εισάγει σωστά τα στοιχεία σας.

Πώς δημιουργώ μια νέα λίστα;

Βήμα 1: Μετά τη σύνδεσή σας, πηγαίνετε στη σελίδα "Οι Λίστες μου".

Βήμα 2: Κάντε κλικ στο κουμπί "Νέα Λίστα".

Βήμα 3: Δώστε έναν τίτλο στη λίστα σας.

Βήμα 4: Επιλέξτε αν η λίστα θα είναι δημόσια ή ιδιωτική:

  • Δημόσια: Θα είναι ορατή σε όλους τους χρήστες που σας ακολουθούν
  • Ιδιωτική: Θα είναι ορατή μόνο σε εσάς

Βήμα 5: Κάντε κλικ στο κουμπί "Δημιουργία" για να αποθηκεύσετε τη λίστα.

Πώς προσθέτω βίντεο σε μια λίστα;

Βήμα 1: Ανοίξτε τη λίστα στην οποία θέλετε να προσθέσετε βίντεο.

Βήμα 2: Κάντε κλικ στο κουμπί "Προσθήκη Βίντεο".

Βήμα 3: Χρησιμοποιήστε το πεδίο αναζήτησης για να βρείτε βίντεο από το YouTube.

Βήμα 4: Επιλέξτε το βίντεο που θέλετε από τα αποτελέσματα αναζήτησης.

Βήμα 5: Κάντε κλικ στο κουμπί "Προσθήκη" για να το προσθέσετε στη λίστα.

Σημείωση: Η αναζήτηση γίνεται απευθείας στο YouTube μέσω του YouTube Data API.

Πώς ακολουθώ άλλους χρήστες;

Βήμα 1: Πηγαίνετε στη σελίδα "Αναζήτηση Χρηστών" ή χρησιμοποιήστε τη γενική αναζήτηση.

Βήμα 2: Αναζητήστε χρήστες βάσει ονόματος, επωνύμου, username ή email.

Βήμα 3: Κάντε κλικ στο προφίλ του χρήστη που θέλετε να ακολουθήσετε.

Βήμα 4: Κάντε κλικ στο κουμπί "Ακολούθηση".

Αποτέλεσμα: Θα μπορείτε να βλέπετε τις δημόσιες λίστες του χρήστη στο δικό σας προφίλ.

Πώς αναπαράγω βίντεο από μια λίστα;

Βήμα 1: Ανοίξτε τη λίστα που περιέχει το βίντεο που θέλετε να παρακολουθήσετε.

Βήμα 2: Κάντε κλικ στον τίτλο του βίντεο ή στο κουμπί "Παρακολούθηση".

Βήμα 3: Το βίντεο θα ανοίξει σε ένα ενσωματωμένο YouTube player.

Επιπλέον επιλογές:

  • Μπορείτε να παρακολουθήσετε όλη τη λίστα διαδοχικά
  • Χρησιμοποιήστε τα κουμπιά πλοήγησης για προηγούμενο/επόμενο βίντεο
Πώς χρησιμοποιώ την αναζήτηση;

Η πλατφόρμα προσφέρει προηγμένες επιλογές αναζήτησης:

Αναζήτηση βάσει κειμένου:

  • Αναζητήστε στους τίτλους των λιστών
  • Αναζητήστε στους τίτλους των βίντεο

Αναζήτηση βάσει ημερομηνίας:

  • Ορίστε εύρος ημερομηνιών (από - έως)
  • Βρείτε περιεχόμενο που προστέθηκε σε συγκεκριμένη περίοδο

Αναζήτηση βάσει χρήστη:

  • Αναζητήστε βάσει ονόματος, επωνύμου, username ή email
  • Βρείτε περιεχόμενο από συγκεκριμένους χρήστες
Πώς αλλάζω το θέμα της σελίδας (Light/Dark);

Βήμα 1: Κάντε κλικ στο κουμπί με το εικονίδιο φεγγαριού (🌙) ή ήλιου (☀️) στο μενού πλοήγησης.

Βήμα 2: Το θέμα θα αλλάξει αμέσως μεταξύ φωτεινού (Light) και σκοτεινού (Dark).

Αυτόματη αποθήκευση: Η προτίμησή σας αποθηκεύεται αυτόματα στον browser σας και θα εφαρμοστεί σε όλες τις επόμενες επισκέψεις.

Διαθέσιμα θέματα:

  • Light Theme: Φωτεινό θέμα με λευκό φόντο
  • Dark Theme: Σκοτεινό θέμα με μαύρο φόντο
Πώς επεξεργάζομαι το προφίλ μου;

Βήμα 1: Μετά τη σύνδεσή σας, πηγαίνετε στη σελίδα "Το Προφίλ μου".

Βήμα 2: Κάντε κλικ στο κουμπί "Επεξεργασία Προφίλ".

Βήμα 3: Τροποποιήστε τα στοιχεία που θέλετε να αλλάξετε.

Βήμα 4: Κάντε κλικ στο κουμπί "Αποθήκευση" για να επιβεβαιώσετε τις αλλαγές.

Διαγραφή λογαριασμού: Αν θέλετε να διαγράψετε τον λογαριασμό σας, υπάρχει σχετική επιλογή στη σελίδα επεξεργασίας προφίλ. Προσοχή: Αυτή η ενέργεια διαγράφει όλα τα στοιχεία σας και τις λίστες σας και δεν μπορεί να αναιρεθεί.

Συχνά Προβλήματα & Λύσεις

Πρόβλημα: Δεν μπορώ να συνδεθώ στον λογαριασμό μου

Λύση: Ελέγξτε ότι έχετε εισάγει σωστά το username/email και τον κωδικό πρόσβασης. Βεβαιωθείτε ότι το Caps Lock είναι απενεργοποιημένο.

Πρόβλημα: Το βίντεο δεν παίζει

Λύση: Ελέγξτε τη σύνδεσή σας στο διαδίκτυο. Το βίντεο μπορεί να έχει αφαιρεθεί από το YouTube ή να μην είναι διαθέσιμο στη χώρα σας.

Πρόβλημα: Δεν βλέπω τις λίστες ενός χρήστη που ακολουθώ

Λύση: Ο χρήστης μπορεί να έχει μόνο ιδιωτικές λίστες. Μόνο οι δημόσιες λίστες είναι ορατές στους followers.

Πρόβλημα: Η αναζήτηση δεν επιστρέφει αποτελέσματα

Λύση: Δοκιμάστε με διαφορετικές λέξεις-κλειδιά ή ελέγξτε τις ημερομηνίες αναζήτησης.

Χρήσιμες Συμβουλές
  • Οργανώστε τις λίστες σας: Δημιουργήστε θεματικές λίστες (π.χ. "Μουσική", "Εκπαιδευτικά", "Ψυχαγωγία")
  • Χρησιμοποιήστε περιγραφικούς τίτλους: Δώστε στις λίστες σας τίτλους που περιγράφουν το περιεχόμενό τους
  • Ακολουθήστε χρήστες με παρόμοια ενδιαφέροντα: Θα ανακαλύψετε νέο και ενδιαφέρον περιεχόμενο
  • Κάντε τακτικά backup: Χρησιμοποιήστε τη λειτουργία εξαγωγής δεδομένων για να κρατήσετε αντίγραφο των λιστών σας
  • Ελέγξτε τις ρυθμίσεις ιδιωτικότητας: Βεβαιωθείτε ότι οι λίστες σας έχουν τη σωστή ρύθμιση (δημόσια/ιδιωτική)
